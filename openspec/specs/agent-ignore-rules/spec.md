## ADDED Requirements

### Requirement: Файл .clineignore

Репозиторий SHALL содержать файл `.clineignore` с перечнем директорий и файлов, которые агент не читает без явного запроса.

#### Scenario: Наличие файла

- **WHEN** агент обходит файловую систему
- **THEN** файлы и директории из `.clineignore` не включаются в контекст

### Requirement: Игнорирование артефактов сборки

`.clineignore` SHALL включать директории сборки .NET: `bin/`, `obj/`.

#### Scenario: Игнорирование bin и obj

- **WHEN** агент читает контекст проекта
- **THEN** содержимое `bin/` и `obj/` не читается без явного запроса пользователя

### Requirement: Игнорирование бинарных файлов

`.clineignore` SHALL включать бинарные файлы: `*.snk`, `*.png`, `*.svg`, `*.jpg`, `*.ico`.

#### Scenario: Игнорирование бинарных файлов

- **WHEN** агент обходит файловую систему
- **THEN** бинарные файлы указанных расширений не читаются без явного запроса
