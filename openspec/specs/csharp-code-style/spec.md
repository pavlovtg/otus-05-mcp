## ADDED Requirements

### Requirement: Отступы в C# коде

C# код SHALL использовать табы (не пробелы) для отступов.

#### Scenario: Новый C# файл

- **WHEN** разработчик создаёт новый C# файл
- **THEN** отступы внутри блоков кода используют символ таба (`\t`)

#### Scenario: Вложенные блоки

- **WHEN** код содержит вложенные блоки (методы, условия, циклы)
- **THEN** каждый уровень вложенности добавляет один таб

### Requirement: Именование в C# коде

C# код SHALL следовать стандартным соглашениям именования .NET:

- Классы, методы, свойства, публичные поля: `PascalCase`
- Локальные переменные, параметры: `camelCase`
- Приватные поля: `_camelCase` (с префиксом `_`)
- Константы: `PascalCase`
- Интерфейсы: `IPascalCase` (с префиксом `I`)

#### Scenario: Публичный метод

- **WHEN** разработчик объявляет публичный метод
- **THEN** имя метода написано в `PascalCase`

#### Scenario: Приватное поле

- **WHEN** разработчик объявляет приватное поле класса
- **THEN** имя поля начинается с `_` и написано в `camelCase`

### Requirement: Структура C# файла

Каждый C# файл SHALL содержать не более одного публичного типа верхнего уровня. Имя файла MUST совпадать с именем публичного типа.

#### Scenario: Публичный класс

- **WHEN** файл содержит публичный класс `MyService`
- **THEN** файл называется `MyService.cs`
