## Requirements

### Requirement: Инструкция по подключению Cline к MCP-серверу через глобальный конфиг

`README.md` SHALL содержать инструкцию по ручному подключению Cline VS Code расширения к MCP-серверу через глобальный конфиг машины. Cline VS Code расширение не поддерживает project-level MCP конфиг — настройка выполняется только на уровне машины.

#### Scenario: Инструкция через панель MCP Servers присутствует

- **WHEN** разработчик читает раздел про MCP в `README.md`
- **THEN** находит пошаговую инструкцию подключения через панель Cline MCP Servers (Remote Servers tab)

#### Scenario: Пример конфигурации присутствует

- **WHEN** разработчик читает раздел про MCP в `README.md`
- **THEN** находит пример JSON-конфигурации для добавления в глобальный `cline_mcp_settings.json`

#### Scenario: Ограничение project-level конфига задокументировано

- **WHEN** разработчик читает раздел про MCP в `README.md`
- **THEN** понимает что конфиг не хранится в репозитории и настраивается на каждой машине отдельно
