## ADDED Requirements

### Requirement: Интеграция с wiki-api

MCP-сервер SHALL получать данные о контрактах из wiki-api.

Базовый URL wiki-api SHALL быть конфигурируемым через переменную окружения `WIKI_API_URL`.

#### Scenario: Конфигурация через переменную окружения

- **WHEN** переменная окружения `WIKI_API_URL` задана
- **THEN** MCP-сервер использует указанный URL для обращений к wiki-api

#### Scenario: Недоступность wiki-api

- **WHEN** wiki-api недоступен
- **THEN** MCP-сервер возвращает ошибку с понятным сообщением вместо необработанного исключения
