## ADDED Requirements

### Requirement: Хранение ADR в docs/adr/

ADR-файлы SHALL храниться в папке `docs/adr/`, разбитой на подпапки по доменам; ADR, специфичные для всех доменов (не привязанные к конкретному) — в `docs/adr/general/`.

#### Scenario: Расположение доменного ADR

- **WHEN** создаётся ADR, относящийся к конкретному домену
- **THEN** файл помещается в `docs/adr/<domain>/`

#### Scenario: Расположение общего ADR

- **WHEN** создаётся ADR, применимый ко всем доменам
- **THEN** файл помещается в `docs/adr/general/`

### Requirement: Индексный файл ADR

`docs/adr/README.md` SHALL содержать одну строку на каждый ADR с номером, названием и статусом.

#### Scenario: Обновление индекса

- **WHEN** создаётся новый ADR
- **THEN** в `docs/adr/README.md` добавляется строка с ссылкой на файл

### Requirement: Чтение индекса ADR в начале задачи

Cline SHALL читать `docs/adr/README.md` в начале каждой задачи.

#### Scenario: Начало задачи

- **WHEN** пользователь начинает новую задачу
- **THEN** Cline читает `docs/adr/README.md` для получения актуального списка решений

### Requirement: Язык ADR — русский, компактный

ADR-файлы SHALL быть написаны на русском языке: одно предложение вместо абзаца, без вводных фраз.

#### Scenario: Создание ADR

- **WHEN** агент создаёт ADR
- **THEN** файл написан на русском языке, буллетами, без вводных фраз
