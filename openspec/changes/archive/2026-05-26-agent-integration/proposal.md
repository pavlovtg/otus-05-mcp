## Зачем

MCP-сервер реализован и запускается через Docker Compose, но агент Cline не подключён к нему — нет инструкции по настройке подключения. Cline VS Code расширение не поддерживает project-level MCP конфиг: подключение настраивается только через глобальный конфиг на уровне машины.

## Что меняется

- В `README.md` добавляется инструкция по запуску MCP и ручному подключению агента через глобальный конфиг
