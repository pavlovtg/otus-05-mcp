# 0006. Стек wiki-сервиса

- Статус: принят
- Дата: 2026-05-25

## Контекст

- Нужен wiki-сервис с REST API и web-интерфейсом для просмотра swagger-контрактов
- Репозиторий использует C# / .NET (ADR-0003)
- Web-интерфейс должен отображать swagger-контракты без возможности выполнять запросы

## Решение

- Backend: ASP.NET Core (C#, .NET 10)
- Frontend: официальный Docker-образ `swaggerapi/swagger-ui`
- Список контрактов передаётся в Swagger UI через переменную окружения `URLS`
- Развёртывание: Docker Compose (два контейнера: wiki-api на порту 5000, swagger-ui на порту 5100)

## Последствия

- Не требуется писать фронтенд — Swagger UI предоставляет готовый UI
- Список контрактов в Swagger UI статичен — требует перезапуска контейнера при изменении набора файлов
- REST API остаётся динамическим — читает файлы при каждом запросе
- .NET 10 поддерживает формат `.slnx` для solution-файлов
