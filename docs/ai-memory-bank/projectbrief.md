# Project Brief

## Цель проекта

- Разработать MCP-сервер на .NET для интеграции с AI-ассистентами
- Реализовать инструменты и ресурсы, доступные через Model Context Protocol

## Ключевые требования

- Платформа: .NET (C#)
- Протокол: MCP (Model Context Protocol)
- Разработка совместно с AI-ассистентом Cline

## Ограничения

- Репозиторий: otus-05-mcp (учебный проект OTUS)
