# Product Context

## Проблема

- AI-ассистенты не имеют доступа к внешним инструментам и данным без специальной интеграции
- MCP-сервер предоставляет стандартизированный интерфейс для подключения инструментов к AI

## Решение

- MCP-сервер на .NET реализует протокол Model Context Protocol
- Предоставляет tools и resources для AI-ассистентов (Claude, Cline и др.)

## Пользователи

- Разработчики, интегрирующие AI-ассистентов в рабочие процессы
- Учебный контекст: курс OTUS
