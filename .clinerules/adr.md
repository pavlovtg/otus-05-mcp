# Работа с ADR

## Расположение

- `docs/adr/README.md` — индекс всех ADR (одна строка на ADR)
- `docs/adr/<domain>/` — ADR конкретного домена
- `docs/adr/general/` — кросс-доменные ADR

## Правила чтения

- Читай `docs/adr/README.md` в начале каждой задачи
- Детали конкретного ADR — по запросу или при необходимости

## Правила создания ADR

- Файл: `docs/adr/<domain>/NNNN-kebab-title.md`
- Номер: следующий после максимального в индексе
- Язык: русский, буллеты, без вводных фраз
- После создания: добавить строку в `docs/adr/README.md`

## Иммутабельность

- ADR неизменяемы после принятия
- Для пересмотра решения — создать новый ADR со статусом "принят, заменяет ADR-NNNN"
