# Memory Bank

## Расположение

`docs/ai-memory-bank/` — 6 файлов:
- `projectbrief.md` — цели и требования проекта
- `productContext.md` — зачем проект, какую проблему решает
- `activeContext.md` — текущий фокус, активные задачи
- `systemPatterns.md` — архитектурные паттерны и решения
- `techContext.md` — стек, инструменты, зависимости
- `progress.md` — что сделано, что осталось, известные проблемы

## Правила чтения

- Читай все файлы memory bank в начале каждой новой задачи
- Читай `docs/adr/README.md` для получения актуального списка архитектурных решений

## Правила обновления

- Обновляй memory bank при значимых изменениях: новые решения, смена архитектуры, прогресс
- Формат: буллеты, одно предложение, без вводных фраз, русский язык
