# Формат промтов

## Структура промта

Используй формат: **Роль / Контекст / Задача / Требования**

- **Роль** — кто ты в этом контексте (1 строка)
- **Контекст** — минимально необходимый фон (буллеты)
- **Задача** — что нужно сделать (1-2 предложения)
- **Требования** — ограничения и критерии (буллеты)

## Принципы экономии токенов

- Одно предложение вместо абзаца
- Буллеты вместо прозы
- Без вводных фраз ("Конечно!", "Отличный вопрос!")
- Без повторения условия задачи в ответе
- Код без избыточных комментариев
